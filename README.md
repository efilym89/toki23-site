# Токи23

Локальный SPA-проект доставки суши и роллов с управляемой админ-панелью на `localStorage`.

## Запуск

```bash
node server.mjs
```

После запуска откройте:

- `http://localhost:4173/` — витрина
- `http://localhost:4173/admin?section=dashboard` — админ-панель

## Что реализовано

- публичные страницы: главная, каталог, карточка товара, корзина, checkout, акции, FAQ, контакты, кабинет, история заказов, legal-страницы и 404;
- динамическая корзина, модификаторы, промокоды, автоакции, доставка по зонам и оформление заказа;
- админка для управления контентом, категориями, товарами, заказами, зонами, акциями, клиентами и базовыми настройками;
- единое хранилище данных через `localStorage`, из которого читают и витрина, и админка.

## Файлы

- `index.html` — SPA-оболочка;
- `src/app.js` — маршруты, UI и бизнес-логика;
- `src/styles.css` — стили витрины и админки;
- `src/default-state.js` — стартовые данные;
- `server.mjs` — локальный сервер без зависимостей.
